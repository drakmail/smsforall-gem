# Smsforall

Client for smsforall.ru site API (http://www.smsforall.ru/doc/api/)

## Installation

Add this line to your application's Gemfile:

    gem 'smsforall'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install smsforall
